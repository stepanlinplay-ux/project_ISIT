## Тематика сайта, название

Веб-сервис для  загрузки и продвижения ртистов и музыкантов.

Название проекта: SoundBoost

__Суть__ : SoundBoost — это музыкальная платформа без модерации, где любой артист может мгновенно выложить свой трек и сразу начать собирать аудиторию.

## Цель и задачи системы

__Цель__: Создать открытую музыкальную площадку, где артисты могут самостоятельно загружать свои треки без предварительной проверки,
 а слушатели — находить новую музыку и влиять на позиции исполнителей в рейтингах.

__Задачи системы:__

1) Разработать систему регистрации и разграничения ролей («Пользователь» / «Артист»).
2) Реализовать загрузку аудиотреков в форматах MP3, WAV, FLAC.
3) Создать автоматические рейтинги:
   - **Топ месяца** (по прослушиваниям и лайкам)
   - **Топ обложка** (голосование за лучшее визуальное оформление)
   - **Топ новичков** и другие метрики
4) Обеспечить поиск и фильтрацию треков по жанрам, дате, популярности.
5) Добавить возможность комментирования, лайков и добавления в плейлисты.

## Целевая аудитория

1. **Начинающие и независимые артисты**  
   — которым нужна платформа без модерации и бюрократии для быстрого старта.
2. **Слушатели (обычные пользователи)**  
   — которые ищут свежую, неизвестную музыку и хотят участвовать в формировании рейтингов.
3. **Музыкальные блогеры / энтузиасты**  
   — для поиска новых талантов и создания обзоров.

## Содержание сайта

- **Главная страница** — с динамическими блоками «Топ месяца», «Свежие треки», «Лучшие обложки».
- **Личный кабинет** (разный для Артиста и Пользователя).
- **Страница артиста** — с его треками, биографией, обложками и статистикой.
- **Страница трека** — плеер, информация о лайках, комментарии, кнопка «Добавить в плейлист».
- **Страница рейтингов / голосований**.
- **Форма загрузки трека** для артистов.
- **Поиск** по названию, жанру, имени артиста.


## Функциональные возможности

###  Без регистрации (гость)
- Прослушивание треков (возможно с ограничением по времени)
- Просмотр рейтингов и страниц артистов

### Для обычных пользователей (после регистрации)
- Прослушивание треков без ограничений
- Постановка лайков / дизлайков
- Комментирование треков
- Создание и сохранение плейлистов
- Голосование в номинации «Топ обложка месяца»
- Просмотр истории прослушиваний

### Для артистов (специальная роль)
- **Загрузка аудиотреков без проверки** (самомодерация)
- Загрузка обложки для каждого трека
- Редактирование своей страницы (био, фото, соцсети)
- Просмотр статистики по каждому треку (прослушивания, лайки, место в рейтингах)
- Возможность закрепить один трек в топе своей страницы

###  Администратор
- Просмотр жалоб на треки (нарушение авторских прав / контент 18+)
- Блокировка артистов или отдельных треков
- Ручная корректировка рейтингов (в исключительных случаях)
